## Getting Started

### Installing dependencies

```bash
bun install
```

### Run dev server

```bash
bun run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Linter

To run the linter use:

```bash
bun run lint
```
